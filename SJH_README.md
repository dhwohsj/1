# 随机手机号生成器

## 功能特点

### ✨ 核心功能
- 🌍 支持 **239个国家和地区** 的手机号生成
- 📱 生成真实格式的手机号（符合各国运营商规则）
- 🔢 自定义生成数量（1-10000）
- 🔍 强大的搜索功能（支持国家名和区号搜索）
- 📥 一键下载所有生成的手机号到txt文件
- 📋 单个/批量复制功能

### 🎨 用户体验
- 📱 完美适配移动端（触摸反馈、手势优化）
- ⚡ 性能优化（分页处理、虚拟滚动）
- 🎯 信息密度优化（清晰的视觉层级）
- 🌈 现代化UI设计（玻璃拟态、动画效果）
- 🚫 无依赖任何外部服务（纯前端实现）

### 🛡️ 技术特性
- 分页处理避免页面卡死（国家列表30个/页，结果50个/页）
- 去重算法确保生成的手机号不重复
- 使用真实运营商号段前缀
- 符合各国手机号长度和格式规范

## 访问路径

```
/sjh
```

## 使用方法

1. **选择国家**
   - 点击国家选择器
   - 使用搜索框快速查找（支持国家名或区号）
   - 浏览分页列表选择目标国家

2. **设置数量**
   - 输入需要生成的手机号数量（1-10000）
   - 建议单次生成不超过1000个以获得最佳性能

3. **生成手机号**
   - 点击"生成手机号"按钮
   - 等待生成完成（大量生成时会显示进度）

4. **管理结果**
   - **单个复制**：点击任意手机号即可复制
   - **复制全部**：一键复制所有生成的手机号
   - **下载文件**：导出为txt文件，方便保存和使用
   - **分页浏览**：结果超过50个时自动分页

## 手机号格式说明

### 生成规则
1. **运营商前缀**：使用各国真实运营商的号段前缀
2. **号码长度**：严格遵循各国手机号长度标准
3. **格式规范**：符合国际电话号码格式（+区号 手机号）
4. **去重处理**：确保同一批次内不会有重复号码

### 示例

```
中国 (+86)：+86 13812345678
美国 (+1)：+1 415-555-0123
英国 (+44)：+44 7400123456
日本 (+81)：+81 90-1234-5678
韩国 (+82)：+82 10-1234-5678
```

## 维护建议

### 添加新国家
1. 打开 `lib/phoneData.ts`
2. 在 `phoneCountries` 数组中添加新国家数据：
```typescript
{
  id: 'XX',              // 国家代码
  name: '国家名称',
  code: '+XX',           // 国际区号
  flag: '🏴',           // 国旗emoji
  prefixes: ['XX'],      // 运营商号段前缀数组
  length: 10            // 手机号总长度（不含区号）
}
```

### 更新运营商前缀
- 查找各国运营商官网获取最新号段
- 更新对应国家的 `prefixes` 数组
- 确保 `length` 参数准确（影响生成的号码长度）
